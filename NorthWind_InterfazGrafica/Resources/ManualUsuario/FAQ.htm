Preguntas fecuentes

¿Puedo visualizar mas de un informa al mismo tiempo?

Si, lo unico que tiene que hacer es generar cada informe que desee visualizar sin cerrar el anterior, y de este modo poder comprarlos



¿Es posible ver la informacion de dos clientes al mismo tiempo?

Esta opcion no esta incluida en la aplicación, para ver la información de un cliente distinto tendra que selleccionarlo y toda la informacion del cliente anterior sera borrada para mostrar los nuevos datos del cliente seleccionado



¿Puedo guardar un informe?

Si, para guardarlo, una vez el informe este abierto unicamente seleccione el icono de la parte superior izquierda (Exportar). A continuacion se abrira un dialogo donde usted podra elegir la ubicacion del archivo
